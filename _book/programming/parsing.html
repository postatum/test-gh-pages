Parsing

Parsing

Parsers can be found at package amf.client.parse. All parsing methods are asynchronous, and either returns a promise/future as a result, or receives an async handler.

RAML 1.0 parsing

Parse file with future

Java
    AMF.init().get();

    /* Parsing Raml 10 with specified file returning future. */
    final BaseUnit result = new Raml10Parser().parseFileAsync("file://src/main/resources/examples/banking-api.raml").get();
    System.out.println("Expected unit encoding webapi: " + ((Document) result).encodes());

Parse file with handler

Java
    AMF.init().get();

    /* Parsing Raml 10 with specified file and async handler. */
    new Raml10Parser().parseFile("file://src/main/resources/examples/banking-api.raml", new Handler<BaseUnit>() {
      @Override public void success(BaseUnit result) {
        System.out.println("Expected unit encoding webapi: " + ((Document) result).encodes());
      }

      @Override public void error(Throwable exception) {
        System.out.println("Unexpected exception during parsing: " + exception);
      }
    });

Parse string with future

Java
    AMF.init().get();

    /* Parsing Raml 10 with specified content. */
    final BaseUnit result = new Raml10Parser().parseStringAsync(
            "#%RAML 1.0\n" +
            "\n" +
            "title: ACME Banking HTTP API\n" +
            "version: 1.0").get();
    System.out.println("Expected unit encoding webapi: " + ((Document) result).encodes());

Parse string with handler

Java
    AMF.init().get();

    /* Parsing Raml 10 with specified content and async handler. */
    new Raml10Parser().parseString(
            "#%RAML 1.0\n" +
                    "\n" +
                    "title: ACME Banking HTTP API\n" +
                    "version: 1.0", new Handler<BaseUnit>()
            {

                System.out.println("Expected unit encoding webapi: " + ((Document) result).encodes());
              }

              @Override public void error(Throwable exception) {
                System.out.println("Unexpected exception during parsing: " + exception);
              }
            });
