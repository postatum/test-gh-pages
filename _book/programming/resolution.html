Resolution

Resolution

Resolvers can be found at package amf.client.resolve. A resolution pipeline can be optionally specified to run a different set of steps.

Default resolution

Java
    AMF.init().get();

    /* Parsing Raml 10 with specified file returning future. */
    final BaseUnit result = new Raml10Parser().parseFileAsync("file://src/main/resources/examples/resolution/banking-api-unresolved.raml").get();
    assertThat( result                          ).isInstanceOf(Document.class);
    assertThat( ((Document) result).declares()  ).hasSize(1);

    /* Resolve unit with default pipeline. */
    final BaseUnit resolved = new Raml10Resolver().resolve(result);
    assertThat( resolved                          ).isInstanceOf(Document.class);
    assertThat( ((Document) resolved).declares()  ).hasSize(0);

Editing resolution

Java
    AMF.init().get();

    /* Parsing Raml 10 with specified file returning future. */
    final BaseUnit result = new Raml10Parser().parseFileAsync("file://src/main/resources/examples/resolution/banking-api-unresolved.raml").get();
    assertThat( result                          ).isInstanceOf(Document.class);
    assertThat( ((Document) result).declares()  ).hasSize(1);

    /* Resolve unit with editing pipeline. */
    final BaseUnit resolved = new Raml10Resolver().resolve(result, ResolutionPipeline.EDITING_PIPELINE());
    assertThat( resolved                          ).isInstanceOf(Document.class);
    assertThat( ((Document) resolved).declares()  ).hasSize(1);
